# بوت واتساب الإسلامي 🌙

بوت Node.js لواتساب مبني على مكتبة **Baileys** ويقدم تجربة متكاملة بالأوامر داخل واتساب (بدون داشبورد).

## ✨ المميزات
- تنبيهات الأذان لكل الصلوات تلقائيًا.
- إرسال ملف `019--1.mp3` لكل الصلوات.
- إرسال ملف `052-.mp3` للفجر والمغرب.
- إرسال ملف `046--_up_by_muslem.mp3` يوميًا بعد العشاء (افتراضيًا بعد 10 دقائق).
- تفاصيل الركعات والسنن مع كل تنبيه صلاة.
- أذكار الصباح والمساء والصلوات والنوم + مجلس استغفار + أدعية متنوعة.
- إرسال أذكار الصباح تلقائيًا عند **الشروق**، وأذكار المساء تلقائيًا عند **الغروب**.
- خدمات يومية: مواقيت الشروق والمغرب + ورد يومي (جزء يومي).
- إرسال مصحف PDF يوميًا للدردشات المفعّلة.
- إعدادات لكل مجموعة على حدة (مدينة/دولة + تفعيل/تعطيل الخدمات).

## ملفات الصوت المطلوبة
ضع الملفات داخل مجلد `assets/` بهذه الأسماء:

- تم إنشاء ملفات placeholder فارغة تلقائيًا بنفس الأسماء داخل `assets/` لتقدر تستبدلها مباشرة بملفاتك الصوتية.
- `019--1.mp3`
- `052-.mp3`
- `046--_up_by_muslem.mp3`

> بدل الملفات المحلية تقدر تستخدم روابط مباشرة عبر متغيرات `.env`.

## التشغيل
```bash
npm install
cp .env.example .env
npm start
```

> ملاحظة: التشغيل الآن يتم من `index.js` فقط (نقطة الدخول الرسمية للبوت).
