# بوت واتساب الإسلامي الرمضاني 🌙

بوت Node.js لواتساب يقدم تجربة متكاملة بالأوامر داخل واتساب (بدون داشبورد).

## ✨ المميزات
- تنبيهات الأذان لكل الصلوات تلقائيًا.
- إرسال أذان **52** للفجر والمغرب، وأذان **16** لباقي الصلوات.
- تفاصيل الركعات والسنن مع كل تنبيه صلاة.
- أذكار الصباح والمساء والصلوات والنوم + مجلس استغفار + أدعية متنوعة.
- إرسال أذكار الصباح تلقائيًا عند **الشروق**، وأذكار المساء تلقائيًا عند **الغروب**.
- خدمات رمضانية: الإمساك والإفطار + ورد يومي (جزء يومي).
- إرسال مصحف PDF يوميًا للدردشات المفعّلة.
- إعدادات لكل مجموعة على حدة (مدينة/دولة + تفعيل/تعطيل الخدمات).

## أوامر المستخدم
- `.اوامر`
- `.مواقيت`
- `.live`
- `.رمضان`
- `.ورد`
- `.صباح` / `.مساء` / `.صلاة`
- `.نوم` / `.استغفار` / `.ادعية`

## أوامر الإدارة
- `>connect`
- `>connect1`
- `>disconnect`
- `>status`
- `>setcity|City|Country`
- `>toggle|prayer|athkar|quran|ramadan`

## التشغيل
```bash
npm install
cp .env.example .env
npm start
```

## ملاحظات
- يتم جلب المواقيت من API Aladhan.
- التخزين المحلي في `data/connections.json`.
- يمكن إيقاف الأذكار المرتبطة بالشروق/الغروب عبر `ENABLE_SOLAR_ATHKAR=false`.
