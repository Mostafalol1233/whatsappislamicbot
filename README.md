# بوت واتساب الإسلامي 🌙

بوت واتساب متكامل يقدم:
- مواقيت الصلاة اليومية.
- تنبيهات الأذان التلقائية للمجموعات المرتبطة.
- إرسال ملف أذان رقم **16** لكل الصلوات، وملف أذان رقم **52** للفجر والمغرب.
- أذكار الصباح والمساء وأذكار بعد الصلاة.
- أوامر ربط المجموعات وعرض الحالة المباشرة للصلاة القادمة.

## المميزات

### 🕋 مواقيت الصلاة والأذان
- تنبيه عند دخول وقت كل صلاة (الفجر، الظهر، العصر، المغرب، العشاء).
- إرسال معلومات الركعات والسنن الرواتب مع كل أذان.
- أمر `.مواقيت` لعرض مواقيت اليوم لمدينتك.

### 📖 الأذكار والأدعية
- `.صباح` أذكار الصباح.
- `.مساء` أذكار المساء.
- `.صلاة` أذكار ما بعد الصلاة.
- `.دعاء` أدعية متنوعة.
- إرسال تذكير أذكار المساء تلقائيًا يوميًا بعد العشاء (وقت قابل للتعديل من البيئة).

### 🔄 الربط مع المجموعات
- `>connect` لعرض كل مجموعات الحساب.
- `>connect[رقم]` لربط مجموعة بالتنبيهات.
- `>live` لمعرفة الصلاة القادمة والوقت المتبقي.
- `>disconnect` لفك ربط الدردشة الحالية.

## التشغيل

## 1) تثبيت الحزم
```bash
npm install
```

## 2) إعداد البيئة
انسخ الملف:
```bash
cp .env.example .env
```
ثم عدل القيم حسب مدينتك ومسارات ملفات الأذان.

> مهم: ضع ملفي الأذان في المسارات المحددة (أو استخدم روابط مباشرة عبر `ADHAN_16_URL` و `ADHAN_52_URL`).

## 3) تشغيل البوت
```bash
npm start
```
سيظهر QR في الطرفية، امسحه بتطبيق واتساب.

## الأوامر

### أوامر الخدمات (.)
- `.اوامر`
- `.صباح`
- `.مساء`
- `.صلاة`
- `.دعاء`
- `.مواقيت`

### أوامر الإدارة (>)
- `>connect`
- `>connect1` (أو أي رقم)
- `>live`
- `>disconnect`
