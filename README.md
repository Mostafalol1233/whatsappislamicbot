# بوت واتساب الإسلامي الرمضاني 🌙

بوت Node.js لواتساب يقدم تجربة متكاملة بدل الداشبورد: كل الإعدادات أصبحت بأوامر داخل واتساب.

## ✨ المميزات
- تنبيهات الأذان لكل الصلوات تلقائيًا.
- إرسال أذان **52** للفجر والمغرب، وأذان **16** لباقي الصلوات.
- تفاصيل الركعات والسنن مع كل تنبيه صلاة.
- أذكار الصباح/المساء/بعد الصلاة بصياغة كاملة.
- خدمات رمضانية: الإمساك والإفطار + ورد يومي (جزء يومي).
- إرسال مصحف PDF يوميًا للدردشات المفعّلة.
- إعدادات لكل مجموعة على حدة (مدينة/دولة + تفعيل/تعطيل الخدمات).

## أوامر المستخدم
- `.اوامر` قائمة الأوامر.
- `.مواقيت` مواقيت اليوم للمدينة المحددة.
- `.live` الصلاة القادمة والوقت المتبقي.
- `.رمضان` الإمساك والإفطار اليوم.
- `.ورد` الجزء اليومي.
- `.صباح` / `.مساء` / `.صلاة` / `.دعاء`.

## أوامر الإدارة
- `>connect` عرض المجموعات.
- `>connect1` ربط مجموعة.
- `>disconnect` فك الربط للدردشة الحالية.
- `>status` عرض إعدادات الدردشة.
- `>setcity|City|Country` ضبط المدينة/الدولة.
- `>toggle|prayer|athkar|quran|ramadan` تشغيل/إيقاف خدمة.

## التشغيل
```bash
npm install
cp .env.example .env
npm start
```
